# Shoes-webpage
StrideX – A modern, fully responsive sneaker landing page built with HTML, CSS, and JavaScript featuring interactive pricing, FAQs, animations, and a clean UI. ⭐

## Github Repo Link 
https://github.com/gauri9368gupta-maker/Shoes-webpage/

## live link
https://gauri9368gupta-maker.github.io/Shoes-webpage/

## ✨ Features

- 📱 Fully Responsive Design
- 🖥️ Mobile, Tablet & Desktop Friendly
- 🍔 Responsive Hamburger Navigation
- 💰 Dynamic Pricing Section
  - Monthly / Annual Billing Toggle
  - Multiple Currency Support (INR, USD, EUR)
- ❓ Interactive FAQ Accordion
- 🎉 Newsletter Subscription Validation
- ✨ Scroll Reveal Animations using Intersection Observer
- 🛍️ Product Showcase Cards
- ❤️ Modern UI with Hover Effects
- ⭐ Testimonials Section
- 🎨 Clean and Minimal Design
